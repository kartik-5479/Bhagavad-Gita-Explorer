# 📖 Bhagavad Gita Explorer

### 🌸 *The Eternal Wisdom for Everyday Life*

<p>
An interactive Bhagavad Gita Explorer built with <b>Python</b>, <b>Streamlit</b>, and <b>Pandas</b> that allows users to browse all chapters, explore Sanskrit shlokas with Hindi and English translations, search verses instantly, and discover daily inspirational teachings.
</p>

# ✨ Features

- 📖 Browse all **18 Chapters** of the Bhagavad Gita
- 📝 View individual Shlokas chapter-wise
- 🕉️ Sanskrit Shlokas
- 🪔 Hindi Translation
- 🌍 English Translation
- 🔍 Smart Search Functionality
- 🎲 Daily Inspirational Random Verse
- 🎨 Clean & Modern User Interface
- 📱 Responsive Streamlit Layout
